setupIntents

Params

intents

@type : Array[string]
Use:
Takes in the intents as a array of strings, and converts them into IntentBitsField

Valid params

Undefined : Converts it into all.
all : returns every intentBit.
dmreactions : Converts to DirectMessageReactions
dm : Converts to DirectMessages
bans : Converts to GuildBans
emojis : Converts to GuildEmojisAndStickers
integrations : Converts to GuildIntegrations
invites : Converts to GuildInvites
members : Converts to GuildMembers
guildmessagereactions : Converts to GuildMessageReactions
guildmessagetyping : Converts to guildmessagetyping
guildmessages : Converts to GuildMessages
presences : Converts to GuildPresences
scheduledevents : Converts to GuildScheduledEvents
voicestates : Converts to GuildVoiceStates
webhooks : Converts to GuildWebhooks
guilds : Converts to Guilds
messagecontent : Converts to MessageContent

Returns

returnIntents

@type Array[IntentBitsField]
Use :
Returns all of the intents requested.

import { setupIntents } from "/dist/discord/client.ts"; // directed from the root dir.
import { Client } from "discord.js";
            
var intents : Array<string> = ["messagecontent", "guilds", "guildmessages", "guildreactions"]
var correctedIntents : Array<IntentBitsField> = [];

correctedIntents = setupIntents(intents);

const client = new Client({
    intents : correctedIntents
})
