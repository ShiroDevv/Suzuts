Suzu : SetUpBot

Params

Username

@type string
Use :
The username used to log into minecraft
Valid param : Any valid minecraft email

Host

@type string
Use:
The server to join
Valid param : Any currently running minecraft servers

password

@type string | undefined
Use :
Password used to log into minecraft (if microsoft account, automatically removed at the start)
Valid param : A minecraft password

auth

@type string
Use :
The authorization method to login to minecraft.
Valid param : "mojang" | "microsoft" | "offline"

Returns

@type any (Dont know mineflayers bot type, decided to return any.) Use :
Returns a usable minecraft bot that logs guild messages
